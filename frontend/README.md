# frontend

Telegram Mini App UI — React + Vite။

| Folder | အကြောင်းအရာ |
|--------|----------------|
| `src/components/` | reusable UI |
| `src/pages/` | route screens |
| `src/hooks/` | e.g. Telegram, order |
| `src/lib/` | small helpers (`userId`) |
| `src/styles/` | global CSS |

Data layer: `@data/...` နှင့် `@data/store/appStore` — Firebase မသုံးပါ။
